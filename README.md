# Khmer Dictionary

A simple and clean **Khmer-English Dictionary** mobile app built with React Native and Expo.

##  Project Description

This project is a **Khmer Dictionary** that allows users to search through **100+ Khmer words** with their English meanings and explanations. 

The app currently includes:
- A clean **Login Screen** using fake authentication (DummyJSON API)
- A powerful **Search functionality** to find Khmer words quickly
- Offline-ready word database (100+ Khmer vocabulary)

The main goal of this project is to help users easily search and learn Khmer words.

##  Features

-  Login with Fake API (DummyJSON)
-  Search Khmer words (English + Khmer)
-  Clean and simple user interface
-  100+ Khmer vocabulary entries
-  Responsive design for mobile
-  Fast search functionality

## Technologies Used

- **React Native** (Expo)
- **JavaScript**
- **DummyJSON** (Fake REST API for login)
- **React Navigation**
- **AsyncStorage** (for saving login state)
